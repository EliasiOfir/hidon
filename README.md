## מבנה כל קובץ
כל קובץ סיפור כולל:
- **תקציר** - סיכום מפורט של הסיפור
- **דמויות מרכזיות** - כל דמות ותפקידה
- **מקומות ומה קרה בהם** - גיאוגרפיה של הסיפור
- **מספרים ומה הם מייצגים** - מספרים חשובים לחידון
- **מילות מפתח** - מונחים ומושגים
- **פסוקים לזיהוי** - פסוקי מפתח עם הטקסט המקורי
