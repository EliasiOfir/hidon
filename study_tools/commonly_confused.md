# סיפורים שמתבלבלים — מדריך לחידון התנ"ך

## איך להשתמש בדף הזה
לכל זוג סיפורים דומים יש טבלת השוואה: מה דומה, מה שונה, ואיך לזכור מה שייך לאיפה.

---

## 1. מרגלים (במדבר י"ג) vs מרגלים (יהושע ב')

| קריטריון | מרגלי משה (במדבר י"ג) | מרגלי יהושע (יהושע ב') |
|----------|----------------------|----------------------|
| **שולח** | משה (בציווי ה') | יהושע |
| **כמה מרגלים** | 12 — אחד מכל שבט | 2 (לא נזכרו בשם) |
| **יעד** | כל ארץ כנען | יריחו בלבד |
| **משך הריגול** | 40 יום | ימים ספורים |
| **מי עזר להם** | אף אחד | רחב הזונה |
| **הדיווח** | 10 — דיבה רעה; 2 (כלב ויהושע) — טובה | חיובי: "כי נתן ה' בידנו את כל הארץ" |
| **תוצאה** | גזירת 40 שנה במדבר | כיבוש יריחו מוצלח |
| **נושא מרכזי** | חוסר אמונה | אמונה ותכנון |

### איך להבדיל:
- **במדבר = 12 מרגלים, כישלון.** מספר 12 = מספר שבטים = בעיה (רוב נגד מיעוט).
- **יהושע = 2 מרגלים, הצלחה.** מספר 2 = פשוט ויעיל = רחב עוזרת.

## 2. מי מריבה בשמות vs מי מריבה בבמדבר

| קריטריון | מי מריבה ראשונה (שמות י"ז) | מי מריבה שנייה (במדבר כ') |
|----------|--------------------------|--------------------------|
| **מקום** | רפידים | קדש (מדבר צין) |
| **שלב בנדודים** | תחילה — לפני מתן תורה | סוף — שנה 40 |
| **ציווי ה'** | הכה בסלע | דברו אל הסלע |
| **מה משה עשה** | הכה בסלע (כפי שצווה) | הכה בסלע פעמיים (במקום לדבר) |
| **תוצאה** | מים יצאו, אין עונש | מים יצאו, אבל משה ואהרן נענשו |
| **עונש** | אין | משה ואהרן לא ייכנסו לארץ |
| **שם המקום** | מסה ומריבה | מי מריבת קדש |
| **אירוע נוסף באזור** | מלחמת עמלק (שמות י"ז) | מות מרים (במדבר כ') |

### איך להבדיל:
- **שמות = רפידים = הכאה = בסדר.** הכל "ר": רפידים, ראשונה, ריגש (הכאה).
- **במדבר = קדש = דיבור = חטא.** "ק" = קדש, קצף (משה כעס), קנס (עונש).
- **טריק:** בשמות, משה עשה מה שנצטווה. בבמדבר, משה לא עשה מה שנצטווה.

## 3. ברית בין הבתרים (בראשית ט"ו) vs ברית מילה (בראשית י"ז)

| קריטריון | ברית בין הבתרים (ט"ו) | ברית מילה (י"ז) |
|----------|---------------------|-----------------|
| **סוג הברית** | חזון / טקס בעלי חיים | מצוות מילה |
| **הבטחה מרכזית** | הארץ — "לזרעך נתתי את הארץ" | זרע — "והרביתי אותך במאוד מאוד" |
| **סימן הברית** | אש עוברת בין הבתרים | מילה בבשר |
| **גיל אברהם** | לא מצוין במפורש (לפני 86) | 99 |
| **שינוי שם** | לא | כן: אברם→אברהם, שרי→שרה |
| **נבואה על העתיד** | 400 שנה גלות ושעבוד | לא |
| **בעלי חיים** | עגלה, עז, איל, תור, גוזל | לא |
| **פעולת אברהם** | פסיבי (תרדמה נפלה עליו) | אקטיבי (מל את עצמו ואת ביתו) |

### איך להבדיל:
- **ט"ו = בתרים = ארץ.** ב.ת.ר = בתרים, תנובת ארץ, ריבוי.
- **י"ז = מילה = זרע.** שינוי בגוף = שינוי בשם = שינוי בעתיד.

## 4. אברהם ואבימלך (בראשית כ') vs יצחק ואבימלך (בראשית כ"ו)

| קריטריון | אברהם ואבימלך (כ') | יצחק ואבימלך (כ"ו) |
|----------|-------------------|-------------------|
| **מי אמר "אחותי"** | אברהם על שרה | יצחק על רבקה |
| **מקום** | גרר | גרר |
| **איך התגלתה האמת** | ה' בא לאבימלך בחלום | אבימלך ראה מהחלון שיצחק "מצחק" עם רבקה |
| **האם נלקחה** | כן — שרה נלקחה לבית אבימלך | לא — רבקה לא נלקחה |
| **תגובת אבימלך** | כעס, שאל "מה עשית לנו?" | כעס, אמר "מה זאת עשית לנו?" |
| **פיצוי** | כסף ומתנות לאברהם | אין פיצוי, אבל ברית שלום |
| **אירוע נוסף** | — | ריב בארות (עשק, שטנה, רחובות) |
| **ברית** | ברית בבאר שבע (בפרק כ"א) | ברית בבאר שבע (בפרק כ"ו) |

### איך להבדיל:
- **אברהם = חלום = שרה נלקחה.** אבימלך חלם = התערבות אלוהית ישירה.
- **יצחק = חלון = רבקה לא נלקחה.** אבימלך ראה מחלון = גילוי אנושי.
- **טריק:** בשני המקרים המקום הוא גרר, ואבימלך כועס. ההבדל: חלום vs חלון, נלקחה vs לא נלקחה.

## 5. הגר בורחת (בראשית ט"ז) vs הגר מגורשת (בראשית כ"א)

| קריטריון | הגר בורחת (ט"ז) | הגר מגורשת (כ"א) |
|----------|-----------------|-------------------|
| **סיבה** | שרי עינתה אותה, הגר ברחה מעצמה | שרה דרשה גירוש, אברהם שילח |
| **יוזמה** | הגר (בריחה) | שרה דרשה, ה' אישר (גירוש) |
| **שלב** | בהריון עם ישמעאל | ישמעאל כבר ילד/נער |
| **מקום המפגש עם מלאך** | עין מים במדבר, על דרך שור | מדבר באר שבע |
| **מסר המלאך** | "שובי אל גברתך" — חזרי | לא נאמר לחזור |
| **הבטחה** | "הרבה ארבה את זרעך" — ישמעאל יהיה פרא אדם | "כי לגוי גדול אשימנו" |
| **מה קרה למים** | מצאה עין מים | ה' פקח עיניה — ראתה באר |
| **שם מקום** | באר לחי רואי | באר שבע |
| **תוצאה** | חזרה לבית אברהם, ילדה את ישמעאל | נשארה במדבר, ישמעאל גדל שם |

### איך להבדיל:
- **ט"ז = בריחה = חזרה.** הגר בורחת מרצון ומוחזרת.
- **כ"א = גירוש = נשארה.** הגר מגורשת בעל כורחה ולא חוזרת.
- **טריק:** ט"ז = טרם ישמעאל (בהריון). כ"א = כבר אחרי (ישמעאל נער).

## 6. כיבוש יריחו (יהושע ו') vs כיבוש העי (יהושע ח')

| קריטריון | כיבוש יריחו (ו') | כיבוש העי (ח') |
|----------|-----------------|----------------|
| **גודל העיר** | עיר גדולה ובצורה | עיר קטנה |
| **אסטרטגיה** | ניסית — הקפה 7 ימים, תרועה, חומות נפלו | צבאית — מארב מאחורי העיר |
| **תפקיד הצבא** | הקפה בלבד, לא נלחמו עד שהחומות נפלו | מארב + התחזות לבריחה |
| **נס** | חומות נפלו מעצמן | אין נס מפורש (מלבד הציווי) |
| **חרם** | חרם מוחלט — הכל לה' | חרם חלקי — שלל ובהמות מותרים |
| **ניצולים** | רחב ומשפחתה | אין ניצולים מהעיר |
| **בעיה לפני הכיבוש** | אין | חטא עכן — מפלה ראשונה |
| **ניסיון ראשון** | הצלחה מיידית | כישלון (בגלל עכן), אחרי תיקון — הצלחה |

### איך להבדיל:
- **יריחו = נס = חרם מוחלט.** הכל מה', הכל לה'.
- **העי = מארב = חרם חלקי.** אסטרטגיה אנושית, שלל מותר.
- **טריק:** יריחו באה ראשונה ויש בה נס גדול. העי שנייה ויש בה בעיה (עכן) שצריך לפתור קודם.

## 7. חטא המרגלים (במדבר י"ד) vs חטא עכן (יהושע ז')

| קריטריון | חטא המרגלים (במדבר י"ד) | חטא עכן (יהושע ז') |
|----------|------------------------|-------------------|
| **מי חטא** | כל העם (בעקבות 10 מרגלים) | אדם אחד — עכן בן כרמי |
| **מהות החטא** | חוסר אמונה, סירוב להיכנס לארץ | גניבת חרם — לקח מהשלל |
| **גורם** | פחד מהעממים | חמדנות — אדרת שנער, כסף, זהב |
| **עונש** | קולקטיבי — 40 שנה במדבר, דור שלם מת | קולקטיבי ואז אישי — מפלה בעי, אח"כ עכן נסקל |
| **מי ניצל** | כלב ויהושע | כל ישראל (אחרי הסרת החרם) |
| **איך התגלה החטא** | דיבת המרגלים הייתה פומבית | גורלות — שבט→משפחה→בית→איש |
| **תשובה** | העם ניסו להעפיל — נכשלו | עכן הודה אחרי הגורל |
| **תגובת המנהיג** | משה התפלל — ה' סלח אך גזר עונש | יהושע קרע בגדיו, נפל על פניו |
| **עיקרון** | חטא ציבורי = עונש ציבורי | חטא יחיד = פגיעה בכלל |

### איך להבדיל:
- **מרגלים = ציבורי = חוסר אמונה = מדבר.** רבים חטאו, רבים נענשו.
- **עכן = יחיד = חמדנות = ארץ.** אחד חטא, כולם נפגעו, עד שהיחיד נענש.
- **עיקרון משותף:** שניהם מלמדים שחטא — אפילו של יחיד — משפיע על הכלל.
