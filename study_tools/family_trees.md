# אילנות יוחסין — מדריך לחידון התנ"ך

---

## 1. משפחת אברהם

```
                         תרח
                     ┌────┼────────┐
                   אברם   נחור     הרן
                     │      │        │
                     │    בתואל     לוט
                     │      │     ┌──┴──┐
                     │    רבקה   מואב  בן-עמי
                     │             (מואב) (עמון)
       ┌─────────────┼───────────────┐
  אברהם+שרה    אברהם+הגר      אברהם+קטורה
       │             │               │
     יצחק        ישמעאל         זמרן, יקשן,
       │          (12 נשיאים)    מדן, מדין,
  יצחק+רבקה                    ישבק, שוח
   ┌───┴───┐
  עשיו   יעקב
```

### פרטים חשובים:
- **אברהם** נולד: אברם, שינוי שם בגיל 99 (בראשית י"ז)
- **שרה** נולדה: שרי, שינוי שם באותה ברית (בראשית י"ז)
- **יצחק** נולד כשאברהם בן 100 ושרה בת 90 (בראשית כ"א)
- **ישמעאל** נימול בגיל 13, אברהם נימול בגיל 99 (בראשית י"ז)
- **עשיו ויעקב** — תאומים, עשיו הבכור (בראשית כ"ה)
- **לוט** — בן הרן, אחיין אברהם. יצא עם אברהם מחרן (בראשית י"ב)

### קשרי נישואין:
| זוג | מקור | הערות |
|-----|------|-------|
| אברהם + שרה | בראשית י"ב | שרה גם אחותו (בראשית כ') |
| אברהם + הגר | בראשית ט"ז | שפחת שרה, מצרית |
| אברהם + קטורה | בראשית כ"ה | אחרי מות שרה |
| יצחק + רבקה | בראשית כ"ד | בת בתואל, אחות לבן |
| נחור + מלכה | בראשית כ"ב | בת הרן |

## 2. משפחת משה

```
              לוי
               │
             קהת
               │
            עמרם + יוכבד (בת לוי)
          ┌────┼────┐
        מרים  אהרן  משה
               │      │
         אהרן+אלישבע  משה+ציפורה
          ┌──┬──┬──┐    ┌───┴───┐
        נדב אביהוא  גרשום  אליעזר
             אלעזר
             איתמר
               │
            אלעזר
            (כהן גדול
            אחרי אהרן)
```

### פרטים חשובים:
- **יוכבד** — בת לוי, אשת עמרם (שמות ב')
- **מרים** — האחות הגדולה, שמרה על משה ביאור (שמות ב')
- **אהרן** — דובר משה, כהן גדול ראשון
- **ציפורה** — בת יתרו/רעואל כהן מדין (שמות ב')
- **גרשום** — בן משה הבכור, שמו: "כי גר הייתי בארץ נכריה" (שמות ב')
- **אליעזר** — בן משה השני (שמות י"ח)
- **אלעזר בן אהרן** — ירש את הכהונה הגדולה אחרי מות אהרן בהר ההר (במדבר כ')

### שושלת כהונה:
```
אהרן (כהן גדול ראשון)
  │
  └──→ אלעזר (כהן גדול שני, במדבר כ')
```

## 3. יהושע וכלב

```
יהושע בן נון
  │
  ├── שבט: אפרים
  ├── תפקיד: משרת משה → מנהיג ישראל
  ├── שם מקורי: הושע בן נון (במדבר י"ג)
  └── משה שינה שמו ליהושע

כלב בן יפונה
  │
  ├── שבט: יהודה
  ├── תפקיד: מרגל, נאמן יחד עם יהושע
  └── כינוי: הקנזי
```

### יהושע וכלב — מה שונה ומה דומה:
| נושא | יהושע | כלב |
|------|-------|-----|
| שבט | אפרים | יהודה |
| מספר מרגל | ה' (במדבר י"ג) | הרביעי (במדבר י"ג) |
| תפקיד עתידי | מנהיג כל ישראל | מקבל חברון כנחלה |
| עמדה בחטא המרגלים | התנגד לדיבה | התנגד לדיבה |
| עונש 40 שנה | ניצל | ניצל |

## 4. שופטים — מג':ז' עד ט'

| # | שופט | שבט | אויב | ספר ופרק | פרטים עיקריים |
|---|------|-----|------|----------|--------------|
| 1 | **עתניאל בן קנז** | יהודה | כושן רשעתיים מלך ארם נהריים | שופטים ג':ז'-י"א | אחי כלב הקטן, שופט ראשון, 40 שנות שקט |
| 2 | **אהוד בן גרא** | בנימין | עגלון מלך מואב | שופטים ג':י"ב-ל' | איטר יד ימין, הרג את עגלון בחרב קצרה |
| 3 | **שמגר בן ענת** | לא צוין | פלשתים | שופטים ג':ל"א | הרג 600 פלשתים במלמד הבקר |
| 4 | **דבורה** (+ ברק בן אבינועם) | אפרים / נפתלי | יבין מלך חצור, סיסרא שר צבאו | שופטים ד'-ה' | נביאה ושופטת, ברק הוביל את הצבא, יעל הרגה את סיסרא |
| 5 | **גדעון** (ירובעל) | מנשה | מדין, עמלק, בני קדם | שופטים ו'-ח' | ניפוץ כדים ולפידים, 300 לוחמים, גיזת הצמר |
| 6 | **אבימלך בן גדעון** | מנשה | מרד פנימי | שופטים ט' | לא שופט אמיתי — מלך, הרג 70 אחיו, משל יותם, מת בתבץ |

### מחזור השופטים:
```
   ① בני ישראל עושים הרע בעיני ה'
                 ↓
   ② ה' מוסרם ביד אויב (שעבוד)
                 ↓
   ③ בני ישראל צועקים אל ה'
                 ↓
   ④ ה' מקים שופט — מושיע
                 ↓
   ⑤ שקט בארץ X שנים
                 ↓
   ⑥ השופט מת → חוזר ל-①
```

### שנות שעבוד ושקט:

| שופט | שנות שעבוד | שנות שקט |
|------|-----------|---------|
| עתניאל | 8 שנים (כושן רשעתיים) | 40 שנה |
| אהוד | 18 שנים (מואב) | 80 שנה |
| שמגר | — | — |
| דבורה | 20 שנים (יבין) | 40 שנה |
| גדעון | 7 שנים (מדין) | 40 שנה |
| אבימלך | — (מלך 3 שנים) | — |

### משל יותם (שופטים ט'):
```
הזית סירב → התאנה סירבה → הגפן סירב → האטד (קוץ) הסכים למלוך
                                                    │
                                              = אבימלך
```
משמעות: מי שראוי למלוך לא רוצה, ומי שלא ראוי — רוצה.
